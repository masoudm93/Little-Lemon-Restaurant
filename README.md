## Quick Start

```shell
$ git clone https://github.com/masoudm93/Little-Lemon-Restaurant.git
$ cd Little-Lemon-Restaurant
$ npm install
```

### Launch the app

Use the following command to run the app in your localhost.

```
npm start
```
